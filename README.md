# tdd-fundamentals-typescript

tdd-fundamentals-typescript contains some katas for test-driven development with TypeScript.

You'll need [Node.js](https://nodejs.org) and [bun](https://bun.sh).

## Installation

```bash
cd rock-paper-scissors
bun install
```

## Usage

```bash
bun run test
```
